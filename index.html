Ethos

This project is intended to house the most up-to-date information regarding my campaign world. I want to focus on hosting as much content as possible, not making the project more complicated than it must be to hold campaign materials.

Goals

  • Use javascript to handle the nav bar with absolute paths
  • Create workable template for new towns
  • Create workable template for NPCs
  • Create workable template for Player Characters
  • Figure out where deities go
  • Figure out where organizations go
  • Clickable map!!!
